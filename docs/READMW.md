# Презентация \- MovieCatalog (Этап 1)

Папка `presentation-stage1` содержит статическую презентацию на Reveal.js.

Как развернуть на GitHub Pages:
1. Скопировать содержимое папки в репозиторий. Поместить в `docs/` или в корень.
2. В репозитории GitHub зайти в Settings → Pages и выбрать ветку `main` и папку `/docs` (или `root` если положили в корень).
3. Подождать публикации — сайт будет доступен по указанному URL GitHub Pages.

Альтернативно можно использовать branch `gh-pages` и инструмент `gh-pages` для деплоя.
